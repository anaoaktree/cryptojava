4.2 - Using X509 certificates

As chaves públicas do protocolo Station-to-station necessitam de ser certificadas por autoridades de confiança (CA). Neste exercício vamos basear-nos em certificados X509.

Obter certificados e chaves privs pubk do servidor, priv servidor, pub cliente, priv cliente autoridade de certificação; Programa Java para validação

  1. Após compilar, correr java ValidateCertPath cacert.pem server_cert.pem para verificar a validade do servidor ou java ValidateCertPath cacert.pem client_cert.pem para verificar a validade do cliente.
