1.1 - Simple client-server application

Este exercício tem como objetivo a preparação do ambiente de trabalho para o resto dos trabalhos. Para isso, vai ser montada uma aplicação cliente-servidor com trocas de mensagens muito simples.

Get Java Client and Server

  1. Depois de compilar os ficheiros, correr o servidor com java Server;

  2. Correr o cliente com java Client;

  3. O cliente abre uma interface de escrita. Tudo o que é escrito é impresso no servidor com o id do cliente.

  4. O servidor fica à escuta de mensagens dos clientes, que imprime no ecrã.
  5. Para terminar o cliente, escrever "exit".

1.2 - Cifra de ficheiro (utilizando a cifra RC4 da JCA/JCE)

Get Rc4.java; cifrar_rc4, gerar_chave and decifrar_rc4 scripts.

  • Pré-requisitos: ter um ficheiro texto_limpo (sem extensões) na diretoria.
  • Gerar chave: escrever uma chave no ficheiro chave ou correr ./gerar_chave;
  • Cifrar: correr ./cifrar_rc4 utiliza os ficheiros chave e texto_limpo e cifra para o ficheiro criptograma;
  • Decifrar: correr ./decifrar_rc4 utiliza os ficheiros chave e criptograma e decifra para o ficheiro texto_decifrado;

Alternativa

  • java Rc4 -genkey chave
  • java Rc4 -enc chave texto_limpo criptograma
  • java Rc4 -dec chave criptograma texto_decifrado

1.3 - Implementação própria da cifra RC4 (Arc4.java)

Get Java Arc4

Utilizando a mesma lógica acima, e depois de complilar o java:

  1. java Arc4 -genkey chave;
  2. java Arc4 -enc chave texto_limpo criptograma;
  3. java Arc4 -dec chave criptograma texto_decifrado;
